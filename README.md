# PizzaDAO's google-apps-scripts
Important Setup Step
Before running these, you must add your secrets to your project:

Click the Gear Icon (Project Settings) on the left sidebar of the Apps Script editor.

Scroll to Script Properties.

Add the following keys and paste your actual values:

TW_CONSUMER_KEY, TW_CONSUMER_SECRET, TW_ACCESS_TOKEN, TW_ACCESS_TOKEN_SECRET

BOT_API_KEY

INTERNAL_PASSWORD
